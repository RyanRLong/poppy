Poppy
=====

Dead simple popup messages

## Current Version
1.1.0

## Usage
Drop the poppy.min.css and poppy.min.js into your page
  <script src="poppy.min.js"></script>

  (function(){
    p = new Poppy();
    p.error("This is an error message");
  })();
